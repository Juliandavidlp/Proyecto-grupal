usar la consola de gitbash:

python -m venv venv
source venv/Scripts/activate


deactivate (para salir del entorno virtual)

para ver las dependencias instaladas en el entorno local
pip freeze 

pip install 

Para instalar las dependencias de flask
pip install flask

en requirements.txt alojamos el contenido del pip freeze:
pip freeze > requirements.txt

Para leer el archivo anterior e instalar las dependencias del mismo:
pip install -r requirements.txt
